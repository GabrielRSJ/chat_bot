## 📋 Sobre o Projeto

chat desenvolvido em python enquanto explorava as diferentes bibiliotecas presentes para python, o codigo esta ligado a api do gemine, então é possivel ter uma conversa e fazer perguntas de diferentes conteudos, a conversa so para quando o usuario escrever 'fim'.

## 🧠 Desafios que enfrentei

- **entender os codigos da biblioteca:** tive que ver o que cada comando fazia ate entender como ligar tudo
- **diferentes versões:** tive que descobrir dentre as varias versões qual era a melhor, permitia varias perguntas por dia


## 🛠️ Tecnologias Utilizadas

- Python 3
- Biblioteca `google.generativeai` (utilizar a ia)
- Biblioteca `userdata` (proteger minha api key)

## 🎯 O que eu aprendi

- ✅ manipular o a biblioteca google.generativeai
- ✅ ligar a api do gemine ao codigo python
